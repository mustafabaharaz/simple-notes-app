## üì± **Usage Guide**

### **Creating and Managing Notes**
1. **New Note**: Click "üìù New Note" or `Ctrl+N`

### **Organization and Sorting**

## üõ°Ô∏è **Security Considerations**

### **Data Privacy**
- **Local Storage**: All data stored locally in browser
- **No External Servers**: No data transmitted to external services
- **Client-Side Encryption**: Encryption performed entirely in browser
- **XSS Protection**: All user content sanitized with DOMPurify

### **Encryption Details**
- **Basic Encryption**: XOR-based encryption suitable for casual privacy
- **Advanced Encryption**: Enhanced encryption patterns for sensitive data
- **Password Requirements**: Strength validation encourages secure passwords
- **Key Management**: Passwords not stored, re-enter required for decryption

### **Data Backup Recommendations**
1. **Regular Exports**: Export to JSON format weekly for complete backup
2. **Multiple Formats**: Use Markdown exports for human-readable backups
3. **External Storage**: Save exports to cloud storage or external drives
4. **Version Control**: Keep multiple export versions for data recovery
